This is the API

getPartsWMV

URL

http://localhost/api/parts/d/{did}/{wvm}/{wvmid}

HTTP Method

GET

Response Type

List

Parameters

  • Path: did String

    Document ID.

  • Path: wvm String

    One of w or v or m corresponding to whether a workspace or version or microversion was entered.

  • Path: wvmid String

    Workspace (w), Version (v) or Microversion (m) ID.

  • Query: elementId String

    Element ID

  • Query: withThumbnails Boolean

    Whether or not to include thumbnails (not supported for microversion)

  • Query: includePropertyDefaults Boolean

    If true, include metadata schema property defaults in response

  • Query: linkDocumentId String

    Id of document that links to the document being accessed. This may provide additional access rights to the document. Allowed only with version (v) path parameter.

  • Query: configuration String

    Configuration string.
